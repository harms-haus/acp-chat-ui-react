Task 2: Fix Vitest Test Discovery Configuration for acp-chat-core
=====================================================================

Date: 2026-04-11
Status: COMPLETED

## Problem Identified

The root vitest.config.ts uses patterns relative to the monorepo root:
  include: ["packages/*/src/**/*.test.ts", ...]

When running `cd packages/acp-chat-core && pnpm vitest run`, these patterns
don't match any files because:
  - Working directory: packages/acp-chat-core
  - Pattern expects: packages/*/src/**/*.test.ts
  - Actual path: src/**/*.test.ts (relative to package directory)

This caused vitest to report: "No test files found, exiting with code 1"

## Solution Implemented

Created package-specific vitest configuration:
  File: packages/acp-chat-core/vitest.config.ts

Configuration:
  - include: ["src/**/*.test.ts", "src/**/*.test.tsx", "src/**/*.spec.ts", "src/**/*.spec.tsx"]
  - globals: true
  - environment: "node"
  - setupFiles: ["../../vitest.setup.ts"] (reference to root setup file)
