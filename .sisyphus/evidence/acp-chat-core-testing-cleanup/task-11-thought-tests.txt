## Key Test Scenarios

### groupThoughtItems()
- Groups consecutive thoughts and tool calls
- Separates groups by messages
- Handles missing timestamps with Date.now() fallback
- Returns empty array when no thoughts/tool calls present

### createGroupedTimeline()
- Creates mixed timeline with messages and thought groups
- Preserves order of items
- Properly closes thought groups when encountering messages

### isThoughtGroupActive()
- Returns true for groups with items created within 5 seconds
- Returns true when isAgentTyping is true (regardless of age)
- Returns false for old groups when not typing
- Checks only the last group in the array

### shouldThoughtGroupBeOpen()
- isActive=true → uses defaultOpenWhenActive
- wasActive+hasBeenActive=true → uses defaultOpenWhenIdle
- Otherwise → uses defaultOpen
- Parameter priority: isActive > wasActive+hasBeenActive > defaultOpen
