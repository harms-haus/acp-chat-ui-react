## Edge Cases Covered

### SSR Environment Handling
- All functions properly detect SSR environment via `isBrowserEnvironment()`
- No-op behavior in SSR (no errors thrown)
- Proper return values (0 for handles, resolved promises)

### Browser API Unavailability
- navigator.clipboard undefined
- ResizeObserver undefined
- requestAnimationFrame/setTimeout undefined
- document undefined (for clipboard fallback)

### Error Handling
- NotAllowedError (clipboard permission denied)
- Generic errors (re-thrown for consumer handling)
- execCommand fallback failures
- Multiple fallback chain failures

### Clipboard Fallback Logic
- Primary: navigator.clipboard.writeText
- Fallback: document.execCommand('copy')
- No fallback mode (strictClipboard)
- execCommand failure handling

## Test Patterns Used

- Vitest test framework
- jsdom environment for browser API simulation
- Mock factory pattern for `isBrowserEnvironment()`
- Property mocking for global objects (navigator, document, ResizeObserver)
- Spy utilities for console.warn verification
