## No Browser API Dependencies

The `getServerSnapshot()` method:
- Does NOT access `window`
- Does NOT access `document`
- Does NOT use `setTimeout` or other timer APIs
- Returns pure JavaScript objects and Maps

## Code Evidence

```typescript
getServerSnapshot(): AcpStoreSnapshot {
  return {
    session: {
      connectionStatus: "disconnected",
      bridgeStatus: "disconnected",
      sessionId: null,
      initialized: false,
      capabilities: null,
    },
    messages: new Map(),
    thoughts: new Map(),
    toolCalls: new Map(),
    timelineOrder: [],
    turnIdToMessageId: new Map(),
    version: 0,
  };
}
```

This is pure JavaScript with no browser dependencies, making it safe for SSR environments.
